GL-Orbit: Project Identity

Project Name: GL-Orbit
Vibe: Community, Bright, Accessible (Glitter/Orbit)
Mission: เป็นศูนย์กลางข้อมูลและตารางฉายสำหรับแฟนคลับซีรีส์ GL ทั่วโลก

Core Values

Reliability: ข้อมูลเวลาฉายต้องแม่นยำ (รองรับ Timezone และ Uncut version)

Accessibility: เข้าถึงลิงก์รับชมได้ง่ายและรวดเร็ว

Community-Centric: ออกแบบมาเพื่อ "ติ่ง" โดยเฉพาะ ไม่ซับซ้อนแต่ครบถ้วน

Current Status

- UI/Frontend: หน้า Public (ซีรีส์, ตารางฉาย, โปรไฟล์) พร้อมใช้งาน
- Authentication: ระบบสมัครสมาชิก, เข้าสู่ระบบ, แก้ไขโปรไฟล์, เปลี่ยนรหัสผ่าน — ทำงานจริง
- Authorization: 2 Roles (ADMIN, USER) พร้อม Route Protection
- Database: เชื่อมต่อ Neon PostgreSQL จริง พร้อม Mock Data
- Admin Panel: UI พร้อม แต่ยังใช้ Mock Data อยู่
