# Soft Heart Orbit Logo Design

## Goal

Create a new GL-Orbit app logo that feels warm, cute, and minimal without becoming visually busy. The logo should work as a standalone app icon for small surfaces such as favicon, PWA icon, and navigation branding.

## Approved Direction

Use the **Soft Heart Orbit** concept:

- A soft heart shape as the central symbol.
- A thin orbit ring wrapping around the heart to connect the mark to the GL-Orbit name.
- One small satellite/star dot on the orbit for a gentle sense of motion.
- A restrained palette based on the existing GL-Orbit theme: coral, lavender, mint, plum, and cream.

## Visual Style

- Warm and friendly, suitable for a GL fan community.
- Minimal and readable at small sizes.
- Soft rounded forms rather than sharp geometric shapes.
- No text inside the icon.
- Avoid excess sparkles, complex gradients, or small details that would disappear in favicon/app-icon sizes.

## Assets to Produce

- Source SVG logo asset.
- Updated raster app icons if replacing the current PWA/app icon assets.
- Integration into existing logo usage where appropriate, while preserving the current GL-Orbit UI style.

## Constraints

- Keep the mark simple enough for 32×32 usage.
- Preserve the existing brand color direction from `src/app.css`.
- Keep Thai/localized UI text unchanged; this work is visual asset replacement only.
- Do not introduce external design dependencies.

## Validation

- Confirm the SVG renders cleanly.
- Confirm generated icon files exist at the expected paths.
- Run the project check command after integration.
