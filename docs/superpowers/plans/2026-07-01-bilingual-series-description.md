# Bilingual Series Description Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add nullable `description_th` and `description_en` columns to the `series` table, expose them in the admin editor, and display the Thai synopsis (with English fallback) on the public series detail page.

**Architecture:** Extend the existing `series` schema with two text columns. Update the public detail query to derive a single `description` field from Thai with English fallback. Update the admin `SeriesCore` type, the admin series editor form, and the admin API create/update endpoints to accept and persist both languages.

**Tech Stack:** SvelteKit 2, Svelte 5, TypeScript, Drizzle ORM 0.43, PostgreSQL (Neon), Tailwind CSS 4.

## Global Constraints
- Use Svelte 5 runes (`$state`, `$derived`, `$props`, `$effect`) only; no `export let`.
- TypeScript `strict: true`, `module: NodeNext` — imports need `.js` extension.
- UI text is Thai.
- Server DB access must use `await getDb()` in SSR contexts.
- No existing test suite; verify via `npm run check`, `npm run build`, and manual browser checks.

### Task 1: Add description columns to the schema and generate migration

**Files:**
- Modify: `src/lib/server/db/schema.ts`
- Create: `drizzle/0012_add_series_descriptions.sql` (generated)
- Update: `drizzle/meta/_journal.json` (generated)

**Interfaces:**
- Consumes: Existing `series` table definition.
- Produces: `series.descriptionTh` and `series.descriptionEn` columns typed as `text` nullable.

- [ ] **Step 1: Edit schema**

Add two columns inside the `series` table object:

```ts
export const series = pgTable('series', {
	id: uuid('id').defaultRandom().primaryKey(),
	studioId: uuid('studio_id').references(() => studios.id),
	titleTh: varchar('title_th', { length: 255 }),
	titleEn: varchar('title_en', { length: 255 }).notNull(),
	descriptionTh: text('description_th'),
	descriptionEn: text('description_en'),
	posterUrl: text('poster_url'),
	status: seriesStatusEnum('status').notNull().default('UPCOMING'),
	deletedAt: timestamp('deleted_at', { withTimezone: true })
});
```

- [ ] **Step 2: Generate migration**
